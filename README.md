# Subak Irrigation System Simulation

Lansing & Kremer (1993) model of Balinese water temple networks, reimplemented with modern Python.

## What is this?

A simulation of Bali's traditional Subak irrigation system, demonstrating how simple local decision rules lead to emergent global coordination. Farmers face a tradeoff between synchronizing planting (reduces pests) and staggering planting (reduces water stress). The simulation shows how optimal patterns self-organize.

## Key Reference

- Lansing, J.S. & Kremer, J.N. (1993). "Emergent Properties of Balinese Water Temple Networks." American Anthropologist 95(1):97-114

## Run

```bash
python3 main.py
```

Outputs: `subak_simulation.html` (interactive visualization)
